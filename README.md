
# Groq-Powered AI Chatbot

A simple command-line AI chatbot built using **LangChain** and **Groq's LLaMA 3.3 (70B)** model.  
The chatbot maintains conversation history and responds intelligently in real time.

---

# Features

- Uses **Groq LLaMA-3.3-70B-Versatile**
- Maintains chat history
- Secure API key input using `getpass`
- Simple and clean CLI interface
- Built with **LangChain (v3 compatible)**

# Tech Stack

- Python
- LangChain
- Groq API
- LLaMA 3.3 (70B)
